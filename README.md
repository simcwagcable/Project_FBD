# Проект: Верстка сайта "REVO Coffee"
## Проект
Этот репозиторий содержит базовый HTML-каркас страницы, созданный по готовому макету кофейного магазина "REVO Coffee".
## Цель
Основная цель работы — освоение принципов семантической HTML5-разметки и создание структуры страницы, пригодной для дальнейшей стилизации и оптимизации.
## Технологии
- HTML5
